

## Installation
---
#### For ease of installation, it is recommended you use pipx instead of pip.

## Using pipx:

    # Install it
    $ pipx install noty

    # Run it
    $ noty

    # You are now ready to use it!

## Without using pipx:

### Using pip:

    # Install it
    $ pip3 install noty

    # Run it
    $ noty

    # You are now ready to use it!


## How to:

    # Run it
    $ noty #If this doesn't work, try reinstalling using the command sudo.
